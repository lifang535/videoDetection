Rim：https://dl.acm.org/doi/pdf/10.1145/3450268.3453521

加了后续两个模块，但是统计车辆及行人数量在函数内部已经做了，考虑用别的模型在后续模块获得更具体的信息

```
if model.config.id2label[label.item()] == "car":
```

### 3. Traffic 汇总
